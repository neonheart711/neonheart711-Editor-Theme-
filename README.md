# neonheart711 Theme for Acode ⚡

![Logo](https://raw.githubusercontent.com/neonheart711/neonheart711-Editor-Theme-/main/icon.jpg)

A clean Neon Hacker style theme for the Acode editor, designed for developers who love a dark interface with vibrant neon highlights.

---

# About 🧠

NeonHeart711 Theme is a custom Ace Editor theme built for Acode.
It focuses on:

- ⚡ Clean neon aesthetic
- 🌑 Pure black background
- 🧩 Minimal distractions
- 💻 Better code readability

# Features 🎨

- Neon Green based UI (#00ff41)
- Smooth syntax highlighting
- Clean gutter and cursor
- Lightweight & fast
- Fully compatible with Acode

---

# Installation ⚙️

1. Open Acode
2. Goto Plugin
3. Search NeonHeart711 Theme me NeonHeart711
4. Install 
5. Go to:
Settings → Editor Theme → neonheart711 (Editor Theme)
